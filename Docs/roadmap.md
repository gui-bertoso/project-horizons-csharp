# Roadmap

## Short term

- clean up placeholder item names, save names, and labels
- finish weapon balancing across classes
- improve HUD, pause, and menu UX
- review player position persistence and overall run state

## Mid term

- give weapon classes clearer gameplay identity
- reduce the size of very large scripts
- revisit drops, progression, and difficulty scaling
- improve hit, damage, and death feedback

## Long term

- consolidate a main procedural generation pipeline
- mature streaming and chunking systems
- expand inventory and progression systems
- multiplayer or state synchronization, if it remains in scope

## Important technical debt

- exported fields that still have no real gameplay effect
- pattern duplication across some scenes
- save/load flow documentation is still shallow
- experimental paths and main playable paths still coexist in the same project
