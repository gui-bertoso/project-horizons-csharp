# core loop

## loop principal

1. explorar
2. coletar recursos
3. processar / craftar / equipar
4. enfrentar risco ou desafio
5. melhorar personagem, base ou acesso ao mundo
6. desbloquear nova camada de exploração

## loops secundários

### progressão de poder
- obter recursos melhores
- construir ferramentas melhores
- acessar áreas mais perigosas
- enfrentar ameaças mais complexas

### progressão de conhecimento
- aprender sistemas do mundo
- entender padrões de spawn, biomas, inimigos e recursos
- dominar rotas eficientes e decisões de risco

### progressão de infraestrutura
- melhorar inventário
- estabelecer bases ou hubs
- automatizar ou acelerar tarefas repetidas

## loop emocional esperado

- curiosidade
- descoberta
- risco
- recompensa
- preparação
- retorno ao desconhecido

## perguntas de validação

cada feature nova precisa responder:

- isso fortalece o loop principal?
- isso adiciona profundidade ou só complexidade?
- isso melhora retenção sem virar trabalho de carteira assinada?
