# Project Horizons (C#)

## Overview

Project Horizons is a systems-driven game project built using Godot with C# integration.

The project focuses on modular design, data-driven systems, and scalable architecture, allowing for future expansion into procedural generation and large-world simulations.

## Key Features

- modular system design
- autoload-based global managers
- data-driven assets using .tres files
- separation between logic and content
- early support for combat systems (enemies, items, projectiles)

## Current Systems

- item system
- enemy system
- projectile system
- achievements system
- settings system

## Vision

To evolve into a scalable, procedural, and performance-oriented game architecture capable of supporting complex simulations and large worlds.
