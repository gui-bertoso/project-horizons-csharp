# glossário

## chunk
unidade básica de divisão do mundo para geração, streaming e persistência.

## seed
valor base usado para gerar conteúdo procedural de forma determinística.

## streaming
carregamento e descarregamento gradual de conteúdo com base em contexto.

## budget por frame
limite de trabalho permitido em um frame para evitar travamentos.

## data-driven
abordagem em que comportamento depende fortemente de dados/configs em vez de lógica hardcoded.

## snapshot
representação capturada do estado atual de um sistema para salvar, sincronizar ou debug.

## culling
remoção de processamento/renderização de elementos irrelevantes no momento.

## lod
níveis de detalhe diferentes conforme distância/importância.
